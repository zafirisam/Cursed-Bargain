# Cursed Bargain

**Cursed Bargain** is a 2D roguelite action game developed in Unity that experiments with player-driven difficulty. Created as an Undergraduate Research Project, it focuses on how voluntary penalties impact gameplay, tension, and decision-making.

## Core Mechanic: The Cursed Bargain
Instead of automatic difficulty scaling, this game uses a risk-and-reward system controlled entirely by the player.
* **The Choice:** You can choose to trade away fundamental stats (such as maximum health, movement speed, or attack speed) to receive an immediate boost to your attack damage.
* **The Consequence:** If you use this mechanic too many times, you will suffer severe debuffs (like losing the ability to dash or having your health cut in half).
* **The Cleansing:** These severe penalties are only removed after you successfully survive long enough to defeat a Boss.

## Key Features
* **Wave Survival:** Fight through procedurally generated enemy hordes that become progressively harder.
* **Modular Codebase:** Built using `ScriptableObjects` to separate data from logic, keeping the architecture clean and scalable.
* **Meta-Progression:** Collect currency during your runs to buy permanent upgrades from the main menu shop.
* **Impactful Combat:** Features visual feedback, particle effects, and knockback to make every hit feel satisfying.

## Tech Stack
* **Engine:** Unity
* **Language:** C#
* **Patterns:** Component-based design, Object Pooling
